## 设置虚拟内存
在运行应用程序时，有时会用到因为内存爆满，而导致运行失败，这个时候我们可以考虑检查一下是否设置了合适的虚拟内存。

### 1. 如何查看应用程序是因为内存不足而导致的问题
1. 打开 Event Viewer
2. 点击 Windows Logs
3. 右键点击 Application， Filter Current Log, 选中Warning和Error。内存不足的信息一般在 Warning里面。
